What this repo is

This page itself is part of the demo: a single, static HTML document that tells the story, with no external dependencies. All of the heavier demo tools run privately on the phone and are not published here.

The aim is to show how quickly a non-developer can go from ‚Äúrough idea‚Äù to a shareable, production-style page with a bit of AI-assisted scaffolding.

How to view locally

On a machine with Python installed:

  • Clone or download the repo.
  • From the repo folder, run: python -m http.server 8000

Then open http://127.0.0.1:8000/ in a browser.

On Android with Termux, the same command works.
